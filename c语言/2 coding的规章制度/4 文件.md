# 文件

- 文件输入输出

  - 使用>和<做重定向

  - ```shell
    //shell
    ./test > 12.out//将产生的文件写入到12.out
    //可以使用more 12.out查看
    //使用cat > 12.out做写入
    ```

    - 使用<表示文件的输出，使用>表示文件的输出

  - 在stdio.h中有FILE

    - ```c
      //打开文件的标准代码
      FILE *fp = fopen("文件名","r");//打开成功返回1，失败返回0
      ```
